# V2 架构文档

本目录保存 `architecture-v2` 分支的中文版 V2 架构文档集合。

## 文档列表

- `architecture-overview.md` — V2 架构目标、模块拓扑与依赖规则的规范说明
- `module-design.md` — 面向后续实现的模块级摘要说明

## 推荐阅读顺序

1. 先阅读 `architecture-overview.md`，理解分支目的与完整的 V2 系统模型。
2. 再阅读 `module-design.md`，掌握未来 `src/` 中正式实现应遵循的模块划分。
3. 然后阅读 `../../mapping/main-to-v2-mapping.zh-CN.md`，了解 `main` 旧结构与 V2 目标之间的关系。
4. 最后阅读 `../../migration/migration-strategy.md` 与 `../../decisions/ADR-001-branch-boundary.md`，再开始任何实现工作。
